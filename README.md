## 🚀 Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/Kritarth22/PORTFOLIO-NEXTJS.git
cd PORTFOLIO-NEXTJS
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page auto-updates as you edit the code.

Other available scripts:

```bash
npm run build   # Production build
npm run start   # Start production server
npm run lint    # Run ESLint
```
